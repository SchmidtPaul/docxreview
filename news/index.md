# Changelog

## docxreview 0.0.0.9000

- Initial development version.
- Added
  [`extract_comments()`](https://schmidtpaul.github.io/docxreview/reference/extract_comments.md)
  to extract comments with paragraph context.
- Added
  [`extract_tracked_changes()`](https://schmidtpaul.github.io/docxreview/reference/extract_tracked_changes.md)
  to extract insertions and deletions with paragraph context.
- Added
  [`extract_review()`](https://schmidtpaul.github.io/docxreview/reference/extract_review.md)
  to combine both into formatted Markdown output.
